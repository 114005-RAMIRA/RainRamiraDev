Explora algunos de mis proyectos destacados, donde he aplicado mis habilidades en desarrollo web para crear soluciones innovadoras y escalables.

Directorio de Peliculas con API de The Movie Database (TMDB)

Este proyecto es un sitio web básico que permite a los usuarios buscar y ver películas utilizando la API de The Movie Database (TMDB). El sitio incluye una barra de navegación con funcionalidad de búsqueda y una sección principal para mostrar resultados. Utiliza JavaScript para interactuar con la API, cargar películas populares y realizar búsquedas.

Sistema POS para Farmacia.

Un sistema de punto de venta diseñado para optimizar las operaciones de una farmacia. Este software proporciona herramientas eficientes para gestionar transacciones, controlar inventario, realizar facturación y mucho más, todo en un entorno intuitivo y fácil de usar.
